# Install

```bash
$ npm i -g ya-oj-cli
```

# Create Solution

```bash
$ oj new 1A --site=codeforces --lang=cpp
```

# Build Solution

```bash
$ oj build
```

# Test Solution

```bash
$ oj test
```

# Login to OJ

```bash
$ oj login
```

# Submit Solution

```bash
$ oj submit
```
