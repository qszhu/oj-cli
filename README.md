# Usage

## Install

```bash
$ npm i -g ya-oj-cli
```

## Login to OJ

```bash
$ oj login --site=cf
```

## Create Solution

```bash
$ oj new 1A --lang=cpp
```

## Build Solution

```bash
$ oj build
```

## Test Solution

```bash
$ oj test
```

## Submit Solution

```bash
$ oj submit
```

# Known Issues

## USACO
* Problems before 2020 requires file IO.

## UVA
* No auto extraction of sample test case yet.
* Problem ID is in the page url, usually different from that on the page.
